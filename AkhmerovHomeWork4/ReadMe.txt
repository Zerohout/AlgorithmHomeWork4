Программа для нахождения решения задачи о ходе коня.

Функции:
- Изменение размера поля производится в классе Program в переменной структуры TechnicalVariables techVar;
- Изменение задержки выполнения операций производится так же в переменной techVar;
- Отображение статистики успешного выполнения задачи (количество перестановок фигуры, 
примерное количество операций, примерное время успешного выполнения задачи)
- Отображение сообщения о невозможности решить задачу.

Примерное время выполнения операций:
- поле 4x5: ~4 сек.
- поле 5x4: ~4 сек.
- поле 5x5: ~3 сек.
- поле 6x6: ~9-10 мин.
